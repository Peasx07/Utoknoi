# U TOK NOI: Intelligent Flood Detection & Warning System

**U TOK NOI (อุทกน้อย)** คือระบบตรวจจับและแจ้งเตือนภัยน้ำท่วมอัจฉริยะ (Smart Flood Detection) ที่พัฒนาขึ้นเพื่อแก้ปัญหาการแจ้งเตือนภัยที่ล่าช้า โดยการนำเทคโนโลยี AI และ IoT มาประยุกต์ใช้ร่วมกับเสาวัดระดับน้ำของกรมชลประทาน เพื่อให้ประชาชนและหน่วยงานรัฐสามารถติดตามสถานการณ์น้ำได้แบบ Real-time และเตรียมรับมือได้ทันท่วงที

---

## ที่มาและความสำคัญ (Background)
จากปัญหาน้ำท่วมที่สร้างความเสียหายให้กับหลายพื้นที่ และระบบการแจ้งเตือนแบบเดิม (Manual) ที่ต้องใช้เจ้าหน้าที่ไปดูระดับน้ำ ปักธง และแจ้งเตือน ซึ่งอาจเกิดความล่าช้าและไม่ครอบคลุม ทีมผู้พัฒนาจึงสร้างระบบที่สามารถ **"อ่านค่าระดับน้ำอัตโนมัติ"** จากเสาวัดระดับน้ำที่มีอยู่เดิม ด้วย Image Processing และส่งข้อมูลขึ้นระบบ Cloud เพื่อวิเคราะห์และแจ้งเตือนทันที

---

## หลักการทำงาน (System Architecture)

ระบบแบ่งการทำงานออกเป็น 3 ส่วนหลัก:

1.  **Sensing & Detection (KidBright μAI):**
    * ใช้บอร์ด **KidBright μAI** ติดตั้งหน้าเสาวัดระดับน้ำ
    * ประมวลผลด้วย **AI Object Detection** เพื่ออ่านค่าความสูงของระดับน้ำจากแถบสีบนเสา
    * ส่งค่าข้อมูลผ่านโปรโตคอล **MQTT** ไปยังตัวกลาง

2.  **Connectivity & Storage (KidBright 32i):**
    * บอร์ด **KidBright 32i** รับค่าจาก MQTT (Broker: HiveMQ)
    * ส่งข้อมูลขึ้น **Google Sheets** เพื่อจัดเก็บข้อมูล (Data Logging)

3.  **Analytics & Visualization (Web & AI):**
    * นำข้อมูลเข้าสู่กระบวนการ **Data Analytics**
    * ทำนายแนวโน้มระดับน้ำ (Forecasting) ด้วยโมเดล **LSTM (Long Short-Term Memory)**
    * แสดงผลระดับน้ำและความเสี่ยงผ่าน **Web Application** ให้ประชาชนดูได้

##  เทคโนโลยีที่ใช้ (Tech Stack)

### Hardware
* **KidBright μAI:** สำหรับประมวลผลภาพ (Edge AI)
* **KidBright 32i:** สำหรับเชื่อมต่อเครือข่ายและส่งข้อมูล
* **Camera & Power Bank:** อุปกรณ์รับภาพและจ่ายไฟ

### Software & Tools
* **AI Model:** Object Detection (Trained with 500 images via Python & Beautiful Soup)
* **Data Forecasting:** LSTM Neural Network (Keras framework)
* **Communication:** MQTT Protocol (HiveMQ)
* **Database:** Google Sheets (via KBG Sheet Plugin)
* **Web Development:** ระบบแสดงผลแจ้งเตือนสถานะ (ปลอดภัย/เฝ้าระวัง/อันตราย)
