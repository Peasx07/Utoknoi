U TOK NOI: Intelligent Flood Detection & Warning System

U TOK NOI (อุทกน้อย) คือระบบตรวจจับและแจ้งเตือนภัยน้ำท่วมอัจฉริยะ ที่พัฒนาขึ้นเพื่อแก้ปัญหาการแจ้งเตือนภัยที่ล่าช้า โดยการนำเทคโนโลยี AI และ IoT มาประยุกต์ใช้ร่วมกับเสาวัดระดับน้ำของกรมชลประทาน เพื่อให้ประชาชนและหน่วยงานรัฐสามารถติดตามสถานการณ์น้ำได้แบบ Real-time และเตรียมรับมือได้ทันท่วงที

ที่มาและความสำคัญ จากปัญหาน้ำท่วมที่สร้างความเสียหายให้กับหลายพื้นที่ และระบบการแจ้งเตือนแบบเดิมที่ต้องใช้เจ้าหน้าที่ไปดูระดับน้ำและปักธงแจ้งเตือน ซึ่งอาจเกิดความล่าช้าและไม่ครอบคลุม ทีมผู้พัฒนาจึงสร้างระบบที่สามารถอ่านค่าระดับน้ำอัตโนมัติจากเสาวัดระดับน้ำที่มีอยู่เดิม ด้วย Image Processing และส่งข้อมูลขึ้นระบบ Cloud เพื่อวิเคราะห์และแจ้งเตือนทันที

หลักการทำงาน ระบบแบ่งการทำงานออกเป็น 3 ส่วนหลัก ดังนี้:

Sensing & Detection (KidBright μAI) ใช้บอร์ด KidBright μAI ติดตั้งหน้าเสาวัดระดับน้ำ ประมวลผลด้วย AI Object Detection เพื่ออ่านค่าความสูงของระดับน้ำจากแถบสีบนเสา และส่งค่าข้อมูลผ่านโปรโตคอล MQTT ไปยังตัวกลาง

Connectivity & Storage (KidBright 32i) บอร์ด KidBright 32i รับค่าจาก MQTT (Broker: HiveMQ) แล้วส่งข้อมูลขึ้น Google Sheets เพื่อจัดเก็บข้อมูล

Analytics & Visualization (Web & AI) นำข้อมูลเข้าสู่กระบวนการ Data Analytics ทำนายแนวโน้มระดับน้ำ (Forecasting) ด้วยโมเดล LSTM และแสดงผลระดับน้ำและความเสี่ยงผ่าน Web Application ให้ประชาชนดูได้

เทคโนโลยีที่ใช้ Hardware:

KidBright μAI สำหรับประมวลผลภาพ

KidBright 32i สำหรับเชื่อมต่อเครือข่ายและส่งข้อมูล

Camera & Power Bank อุปกรณ์รับภาพและจ่ายไฟ

Software & Tools:

AI Model: Object Detection (เทรนด้วยรูปภาพ 500 รูป ผ่าน Python & Beautiful Soup)

Data Forecasting: LSTM Neural Network

Communication: MQTT Protocol

Database: Google Sheets

Web Development: ระบบแสดงผลแจ้งเตือนสถานะ

การทดสอบและแก้ปัญหา เนื่องจากในช่วงพัฒนาเป็นฤดูร้อนทำให้ระดับน้ำในแม่น้ำจริงแห้งขอด ทีมงานจึงจำลองสถานการณ์โดยสร้างตู้จำลอง (Simulation Tank) และเสาวัดระดับน้ำย่อส่วน เพื่อทดสอบระบบการอ่านค่าของ AI ในสภาพแวดล้อมจำลอง และปรับจูนความแม่นยำก่อนนำไปประยุกต์ใช้จริง
